I’m building an automated social graphics system for my basketball stats site (Swish Assistant).

Goal:
- I want to auto-generate “performance cards” from my stats database.
- Each graphic is a template (designed in Canva), and the site will fill in the stats + player photo.
- This first template is a “Player Night” card (big points, full stat line).
- Later there will be multiple templates (player records, team performance, leaders, etc.), so please structure the code to be reusable.

Tech:
- React + TypeScript + Tailwind (Vite-based project).
- Supabase is my backend.
- I will add a blank Canva template PNG at: `public/card-templates/player-night-v1.png` (background only, no numbers, no text, no photo).

I need you to set up a **new area for social card templates** and wire the first one up.

### TASKS

1. **Create a dedicated area for social card templates**
   - Create folder: `src/components/social/` if it doesn’t exist.
   - Create `src/types/socialCards.ts` to hold shared types for social cards.
   - In `src/types/socialCards.ts`, define:

     ```ts
     export type PlayerNightV1Data = {
       player_name: string;
       team_name: string;
       opponent_name: string;

       points: number;
       rebounds: number;
       assists: number;
       steals: number;
       blocks: number;

       fg: string;          // e.g. "10/20"
       three_pt: string;    // e.g. "1/1"
       ft: string;          // e.g. "1/1"
       turnovers: number;   // e.g. 4
       ts_percent: string;  // e.g. "41.1"
       plus_minus: string;  // e.g. "+11" or "41.1"

       home_score: number;
       away_score: number;

       home_logo_url?: string;
       away_logo_url?: string;
       photo_url?: string;
     };

     export type SocialCard = {
       id: string;
       template: string;
       card_type: string;
       data: any;          // will be cast to specific types (e.g. PlayerNightV1Data)
       image_url?: string | null;
       caption?: string | null;
     };
     ```

2. **Implement the Player Night template component**

   - Create `src/components/social/PlayerNightCardV1.tsx`.
   - This component:
     - Accepts `data: PlayerNightV1Data` as props.
     - Renders a fixed 1080×1350 card.
     - Uses the blank Canva PNG as a background: `background-image: url('/card-templates/player-night-v1.png')`.
     - Positions the dynamic bits (photo, big number, stats) over that background.
